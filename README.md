# MiniCat
A small HTTP server based on the Java Servlet standard, similar to Apache Tomcat.<br>
The features implemented are as follows:
* Servlet
* Session
* Cookie
* Listener
* Nio/Bio 
* Http keep-alive
* WebSocket

The supported ContentTypes are as follows:
* application/json
* multipart/form-data
* application/x-www-form-urlencoded
* text/event-stream
